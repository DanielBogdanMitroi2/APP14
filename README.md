# APP14 - Firebase CRUD Estudiantes

Aplicación Android que conecta con Firebase Realtime Database para realizar operaciones CRUD de estudiantes.

## Requisitos previos
1. Android Studio instalado
2. Proyecto Firebase creado (APP14_2025-26)
3. Archivo `google-services.json` descargado

## Instalación
1. Clonar el repositorio
2. Copiar `google-services.json` en la carpeta `app/`
3. Sincronizar Gradle
4. Ejecutar en emulador o dispositivo

## Funcionalidades
- ✅ Crear estudiantes
- ✅ Leer estudiantes (en tiempo real)
- ✅ Actualizar estudiantes
- ✅ Eliminar estudiantes

## Estudiante
- id
- nombre
- email
- edad
